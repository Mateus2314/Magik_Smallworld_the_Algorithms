## How to use

Declare the method in Emacs and call it by .meter_to_cent() in a second command

```magik
 (user)MagikSF> a << {30, 40, 1, 4 , 5}
 (user)MagikSf> a.meter_to_cent()
"Is 250.0000000 cm"
 ```
