## How to use

Declare the class and methods in Emacs and create the object:

```magik
 (user)MagikSF> fifa_ball << ball_class.new( "White", 25.5, "Carbon")
 (user)MagikSF> fifa_ball.show_color()
"White"
(user)MagikSF> fifa_ball.change_color("Blue")
(user)MagikSF> fifa_ball.show_color()
"Blue"
 ```
