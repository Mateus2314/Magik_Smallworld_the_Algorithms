## How to use

Declare the method in Emacs and call it by .method_calculate_a_media() in a second command

```magik
 (user)MagikSF> a << { 100, 100, 80, 80 }
 (user)MagikSf> a.method_calculate_a_media()
"Average is 90!"
 ```
